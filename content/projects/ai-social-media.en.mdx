## Context

This project was developed as part of the KISOME program. The goal: build a faceless, ethical, sustainable content brand operating on a 10-hour/week budget — generating quiz and cozy aesthetic content across TikTok, Instagram, and Facebook using AI tools.

**Tools used:** ChatGPT · FlowithAI / Manus · Midjourney · ElevenLabs · Adobe Premiere · Suno · n8n

**Constraints:** 10 h/week · 60–80 €/month · 100% anonymity

## Production Workflow & Measurement

<img src="/projects/kisome/production-workflow.jpg" alt="Weekly production workflow system" style={{display:'block',margin:'16px auto',maxWidth:'34%'}} />

**Weekly 10-hour system:**

| Day | Task |
|-----|------|
| Mon | Ideas & scripts (8–10 quiz entries, 4–5 cozy prompts) |
| Tue | Asset generation (voiceovers, AI visuals) |
| Wed | Batch editing: cut 6 quiz videos |
| Thu | Scheduling & captions |
| Fri | Community replies & review |

**Season system:** Each season = 4–6 weeks · stable format → adjustments only after season review

**Primary metrics:**
- Quiz: retention, completion rate, comments
- Cozy: saves, loops, mood keywords
- *Avoid vanity metrics (likes, follower count in early phase)*

**Adaptation:** Monthly reviews (not daily) · identify top-performing topics · soft A/B testing on tone, hook type, pacing

## Updated Cozy Concept — YouTube Lounge Radio

<img src="/projects/kisome/cozy-lounge.jpg" alt="YouTube Lounge Radio – cozy window projection concept" style={{margin:'16px 0'}} />

**New content format:** 24/7-style lounge radio videos on YouTube — relaxing evening lounge music generated with Suno, seamlessly looped cozy videos.

**Visual composition:**
- **Foreground:** cozy interior elements (lamp, table, cup, plants, books) — warm colour palette
- **Mid / window surface:** transparent projection layer for viewer messages
- **Background:** city at evening or night — warm window lights, soft glow, subtle movement (rain, fog, clouds)

**Monetisation mechanic — "Special Thanks Window"**

Based on YouTube Super Thanks / paid comments. Supporters donate 1–2 € or more. Their message appears directly in the video — projected as fine text onto the window glass, animated with a soft glow or reflection effect. Messages rotate in a supporter queue.

**Why this works:**
- Creates a community-driven cozy space
- Enables continuous monetisation without ads or product placement
- Encourages repeat visits ("Is my message showing?")
- Fits the Digital Sanctuary philosophy — slow, long watch sessions
- Works well with YouTube's algorithm (high watch time = more reach)

## Automation System — n8n + AI Agents

<img src="/projects/kisome/automation-diagram.jpg" alt="n8n automation system diagram" style={{margin:'16px 0'}} />

### Quiz Agent — Fully Automated Pipeline

**Trigger:** Daily cron → reads content plan (Google Sheets / Notion)

1. **AI generation (LLM node):** Creates question, 4 answers, correct answer, explanation, difficulty mix (20/60/20), video script, Midjourney prompt
2. **Image generation (Midjourney):** Send prompt via Discord → wait for result → download
3. **Video creation:** Option A — Runway or Freepik API generates the short; Option B — local FFmpeg builds the video (text overlays, light motion, optional TTS)
4. **Auto-publishing:** n8n uploads to YouTube Shorts, Instagram Reels, Facebook Reels, TikTok → updates status + sends daily report

### Lounge Radio Agent — YouTube 24/7 Cozy Stream

**Trigger:** Scheduled automation

1. **Music generation (Suno):** Auto-prompt: "relaxing evening lounge" → download track → update playlist
2. **OBS control (WebSocket):** Switch to "LoungeRadio" scene → trigger YouTube livestream start/stop
3. **Video scene components:** Cozy interior + city view loop · music playlist player · text layer projected on window (Special Thanks)
4. **Special Thanks automation:** n8n fetches YouTube comments / Super Thanks → formats supporter names → sends updated text to OBS via WebSocket
