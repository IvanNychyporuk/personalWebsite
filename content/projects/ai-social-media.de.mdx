## Kontext

Dieses Projekt wurde im Rahmen des KISOME-Programms entwickelt. Ziel: eine gesichtslose, ethische und nachhaltige Content-Marke mit einem Budget von 10 Stunden pro Woche aufzubauen — Quiz- und Cozy-Aesthetic-Inhalte für TikTok, Instagram und Facebook mit KI-Tools generieren.

**Verwendete Tools:** ChatGPT · FlowithAI / Manus · Midjourney · ElevenLabs · Adobe Premiere · Suno · n8n

**Vorgaben:** 10 Std./Woche · 60–80 €/Monat · 100 % Anonymität

| Pillar | Beschreibung |

## Aktualisiertes Cozy-Konzept — YouTube Lounge Radio

<img src="/projects/kisome/cozy-lounge.jpg" alt="YouTube Lounge Radio – Cozy-Fenster-Projektionskonzept" style={{margin:'16px 0'}} />

**Neues Inhaltsformat:** 24/7-artige Lounge-Radio-Videos auf YouTube — entspannende Evening-Lounge-Musik, generiert mit Suno; nahtlos geloopte Cozy-Videos.

**Visuelle Komposition:**
- **Vordergrund:** gemütliche Raumelemente (Lampe, Tisch, Tasse, Pflanzen, Bücher) — warme Farbpalette
- **Mittelgrund / Fensterfläche:** transparente Projektionsfläche für Zuschauerkommentare
- **Hintergrund:** Stadt bei Abend oder Nacht — warme Fensterlichter, weiches Glühen, leichte Bewegungen (Regen, Nebel, Wolken)

**Monetarisierungsmechanik — „Special-Thanks-Fenster"**

Basierend auf YouTube Super Thanks / bezahlten Kommentaren. Supporter können 1–2 € oder mehr spenden. Ihre Nachricht erscheint direkt im Video — als feiner Text auf die Fensterscheibe projiziert, mit sanftem Glow oder Reflexions-Effekt animiert. Nachrichten rotieren in einer Supporter-Queue.

**Vorteile dieses Konzepts:**
- Schafft einen community-getriebenen Cozy-Space
- Ermöglicht kontinuierliche Monetarisierung ohne Werbung oder Produktplatzierung
- Regt Zuschauer zu regelmäßigen Rückkehrbesuchen an („Ist meine Nachricht erschienen?")
- Passt perfekt zur Digital-Sanctuary-Philosophie — entspannend, langsam, langes Zuschauen
- Funktioniert hervorragend mit YouTubes Algorithmus (hohe Watchtime = mehr Reichweite)

## Automation System — n8n + KI-Agenten

<img src="/projects/kisome/automation-diagram.jpg" alt="n8n Automationssystem-Diagramm" style={{margin:'16px 0'}} />

### Quiz-Agent — Vollautomatische Pipeline

**Trigger:** Täglicher Cron → liest den Content-Plan (Google Sheets / Notion)

1. **KI-Generierung (LLM-Node):** Erstellt Frage, 4 Antworten, richtige Antwort, Erklärung, Schwierigkeitsmix (20/60/20), Videoskript, Midjourney-Prompt
2. **Bildgenerierung (Midjourney):** Prompt via Discord senden → auf fertiges Bild warten → herunterladen
3. **Videoerstellung:** Option A — Runway- oder Freepik-API erzeugt das Short; Option B — lokaler FFmpeg-Prozess baut das Video (Text-Overlays, leichte Bewegungen, optional TTS)
4. **Automatisches Publishing:** n8n lädt hoch zu YouTube Shorts, Instagram Reels, Facebook Reels, TikTok → aktualisiert Status + sendet täglichen Report

### Lounge-Radio-Agent — YouTube 24/7 Cozy-Stream

**Trigger:** Geplante Automation

1. **Musikgenerierung (Suno):** Auto-Prompt: „relaxing evening lounge" → Track herunterladen → Playlist aktualisieren
2. **OBS-Steuerung (WebSocket):** Wechsel zur Szene „LoungeRadio" → YouTube-Livestream automatisch starten/stoppen
3. **Video-Szenen-Komponenten:** Cozy-Interieur + Stadtblick-Loop · Musik-Playlist-Player · Textlayer auf Fenster projiziert (Special Thanks)
4. **Special-Thanks-Automatisierung:** n8n ruft YouTube-Kommentare / Super-Thanks ab → formatiert Unterstützer-Namen → sendet aktualisierten Text an OBS via WebSocket

## Ergebnis

Ein vollständiges KI-gestütztes Content-System: **Strategie → Content-Architektur → Prompt-Engineering → Asset-Generierung → Stapelproduktion → Auto-Publishing.**

Wichtigste Erkenntnisse:
- Die Zweikanalstrategie schafft algorithmische Diversität bei einheitlicher Produktion
- Gesichtsloses Branding ermöglicht vollständige Anonymität und langfristige Skalierbarkeit
- Die „Special-Thanks-Fenster"-Mechanik verwandelt passives Zuschauen in Community-Teilnahme
- Vollständige n8n-Automation reduziert die wöchentliche Aktivarbeit auf Recherche und Qualitätskontrolle
